# Be The Hero
Aplicação FullStack criada na semana OmniStack 11 da Rocketseat, utilizando Node no backend, ReactJs no frontend
e React Native no mobile.

## BackEnd
### Bibliotecas utilizadas
- Cors
- Express
- Knex
- SQLite3
- Nodemon (em modo desenvolvimento) 

## FronEnd
### Bibliotecas utilizadas
- Axios
- React Router Dom
- React Icons
